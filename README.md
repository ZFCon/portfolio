## What is this?
A simple portfolio website with a lot of features like upload cv, projects screenshot and about you text and more...

## Project Detail
* Python 3
* Django 
* wagtail
* Bootstrap 4

## Run it in local env

```bash
git clone https://github.com/ZFCon/portfolio.git
cd moviesdand
git checkout master

# setup virtualenv
pip install -r requirements.txt

./manage.py runserver
# http://127.0.0.1:8000/
```
